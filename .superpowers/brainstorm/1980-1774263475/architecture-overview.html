Key Design Decisions

✅ What works

  • 50 WebSocket connections = trivial for Node.js
  • ws package already installed
  • httpServer already created (ws attaches to it)
  • AI question generation already works
  • All UI components exist, just need real data

🔨 What needs building

  • WebSocket server with room management
  • Separate /kahoot/host route (fullscreen)
  • QR code generation on host lobby
  • Real player sync (join → answer → score)
  • Better AI prompts (harder, varied questions)
  • Server-side scoring (prevent cheating)
